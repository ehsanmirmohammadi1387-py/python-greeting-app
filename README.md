# 👋 Greeting App with Attitude

A humorous Python terminal app that greets you, gives random compliments, and calculates your birth year. 
But beware — if your name is Ehsan, you're in BIG trouble! 🤡

## 🚀 Features
- Greets you by name
- Gives random compliments from a list
- Calculates your birth year based on your age
- Special (and hilarious) treatment for anyone named "Ehsan"
- Clown detector for users who type numbers instead of names 🤡

## 🛠️ Tech Stack
- Python 3.x
- `random` library
- Pure fun and sarcasm
